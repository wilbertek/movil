# Music Player (Expo Managed)

Proyecto de ejemplo en **Expo Managed + React Native + TypeScript** que reproduce audio con `expo-av`.

## Requisitos
- Node.js LTS
- Expo CLI (se usa vía `npx`)

## Instalación
```bash
npm install
npx expo install expo-av @react-native-community/slider
```

> Nota: Si `npm install` te da warnings de versiones, lo más importante es que `expo install`
> instale las versiones compatibles con tu SDK de Expo.

## Ejecutar
```bash
npx expo start
```

Escanea el QR con **Expo Go** o abre en emulador.

## Playlist
Edita `App.tsx` y cambia `PLAYLIST` con tus URLs MP3/streams.
